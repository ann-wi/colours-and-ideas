# Colours and Ideas

Storage of ideas for my art and list of all pretty colours I can use with CSS - HEX, RGB, colour names.

## Current Features

1. Add colour based on HEX with name.
2. Displays colours with name and HEX property with visual.
3. Renders default colours from API ('https://jsonlint.com/datasets/colors.json')

## Features to be added

1. Remove added colour
2. Edit added colour
3. Filtered search
4. Save own colours to local storage

## Block with ideas (coming soon) will include
